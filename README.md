# Deezer Player
Плеер для локальной музыки и музыки, полученной через Deezer API.
## Стек разработки
* UI - XML
* Работа с сетью:
* DI: Dagger/Hilt
* Архитектура: чистая архитектура, многомодульность
* Навигация: Jetpack Navigation
* Архитектура: MVVM/MVI
* Фоновый плеер: ExoPlayer
